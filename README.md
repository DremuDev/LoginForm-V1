# Simple Login Form (JavaScript)

A basic login form built using **HTML, CSS, and JavaScript**.
This project demonstrates simple **client-side form validation** and user input handling.

## 🚀 Features

* Simple and responsive login form
* Email and password input fields
* Basic JavaScript validation
* Error messages for invalid inputs
* Clean and beginner-friendly code

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
